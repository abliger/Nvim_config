# Nvim\_config

## 常用快捷键

NvimTree

| 命令 | 描述         |
| ---- | ------------ |
| gto  | 打开文件目录 |
| gte  | 切换文件目录 |

***

Markdown

| 命令    | 描述          |
| ------- | ------------- |
| `<C-p>` | 预览 Markdown |

***

code

|命令|描述|
|---|-----|
|glh|查看函数、变量、类、结构体等描述|

***

buffer

|命令|描述|
|----|---|
|`<c-q>`|选中的buff 写入到光标位置|
|`<Tab>`|切换buff|
|`<S-Tab>`|切换buff|
